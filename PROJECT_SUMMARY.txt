⚙️ ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ
═══════════════════════════════════════════════════════════════

Язык:           Python 3.8+
Фреймворк:      FastAPI + Uvicorn
Протокол:       MCP (Model Context Protocol) over SSE
API:            WordPress REST API v2
Аутентификация: WordPress Application Passwords
HTTPS:          Cloudflare Tunnel
Сервис:         systemd
Логи:           journald
Порт:           8000 (localhost only)

═══════════════════════════════════════════════════════════════
📋 ТРЕБОВАНИЯ
═══════════════════════════════════════════════════════════════

Сервер:
  • Ubuntu 20.04+ / Debian 10+
  • 1GB RAM (рекомендуется 2GB)
  • 10GB диск
  • Python 3.8+

WordPress:
  • WordPress 5.6+ (для Application Passwords)
  • REST API включен (по умолчанию)
  • Админ доступ

ChatGPT:
  • ChatGPT Plus ($20/месяц)
  • Доступ к MCP функциям

🛠️ ИНСТРУМЕНТЫ MCP (TOOLS)
═══════════════════════════════════════════════════════════════

1. create_post
   Создает новый пост на WordPress
   Параметры: title, content, excerpt, status

2. update_post
   Обновляет существующий пост
   Параметры: post_id, title, content, excerpt

3. get_posts
   Получает список постов
   Параметры: per_page, page

4. delete_post
   Удаляет пост
   Параметры: post_id
